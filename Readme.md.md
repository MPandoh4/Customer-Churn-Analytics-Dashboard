# **Customer Churn Analysis Dashboard**

## **Project Overview:**



This Power BI dashboard analyzes customer churn patterns and identifies the factors contributing to customer attrition.



The dashboard helps businesses understand which customer segments are most likely to churn and provides actionable insights to improve customer retention.



#### **Tools Used**

Power BI

DAX Measures

Data Visualization

Data Cleaning

#### **Key KPIs**

Total Customers: 50,000

Churned Customers: 14,000

Churn Rate: 28.90%

Average Lifetime Value: 1.44K

#### **Dashboard Insights**

Customers aged 18–25 show the highest churn rate.

Low login frequency customers churn significantly more than highly engaged customers.

Customers with low app usage exhibit the highest churn rates.

Gender-based churn differences are relatively small.

Country-wise churn variation is minimal.

#### **Business Recommendations**

Develop retention campaigns targeting younger customers.

Increase app engagement through rewards and notifications.

Re-engage inactive users through personalized email campaigns.

Monitor churn KPIs regularly using dashboards.
